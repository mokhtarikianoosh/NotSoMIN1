To compile: In Eclipse, create a project and place Main.java in the src folder.

To Run: After proper setup, click Run and choose to run it as a Java Application.

Database access information: database name is employees, user is root and password is the TA's database password since he'll be running it on his system with the correct database. Password configuration can be found on line 18 of Main.java if using Eclipse.
